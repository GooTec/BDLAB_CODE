# 바이오데이터랩 연구 역량 강화 
한동대학교 바이오데이터랩의 연구역량 강화를 위한 교육 문서입니다. 바이오 데이터의 분석, 시각화 그리고 데이터를 이용한 모델링을 위한 코드와 이 코드를 이해하기 위한 설명을 하고자 합니다. 각 문서는 코드를 작성하고 실행 한 담당자에 의해 문서화 되었습니다.
